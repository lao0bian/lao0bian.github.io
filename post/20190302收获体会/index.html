二叉搜索树的应用

这道题真的是折磨了我一个下午吧，本来自己找了一份代码，按照他的思路来构建树，但是没想到找的那份不靠谱，根本就没有用指针，就生写 写出来的，所以自己一狠心，直接代码重新写，这里借鉴了网上的一篇文章，结合我自己的题目修改了一下，然后就ac了。

题目:

图像的灰度直方图是反映一幅图片中各灰度级像素出现的频率与灰度级之间的关系，也就是说，把这幅图片中每一个像素出现的次数统计出来。请根据给出的灰度值矩阵统计出此图的灰度直方图，即统计每一个灰度值出现的频数。

输入格式： 每个测试包含一个测试样例，第一行给出矩阵的行数m和列数n，以空格隔开，以下输入一个灰度值矩阵，灰度行值之间以空格隔开，灰度值取值范围为0-255。
输出格式：

对每个测试用例中的每一个灰度值，以灰度值v:出现频数t的格式在一行中输出该灰度值v及其在矩阵中出现的次数t。要求按灰度值的升序输出,末尾无多余空格。

输入样例：

4 5
1 3 7 9 12
4 4 8 10 12
4 3 8 9 10
1 4 10 12 8

输出样例:

1:2
3:2
4:4
7:1
8:3
9:2
10:3
12:3

思路：

  1. 将输入的数存入到数组里。
  2. 将数组中的数存到二叉搜索树中去。
  3. 要注意树的构建过程中也要存储该点出现了几次（应题目要求）。
  4. 中序遍历按要求输出树即可。

#include<iostream>
const int t =100;
using namespace std;
typedef  struct bstNode    //我查了一下发现typedef 是c语言比较常用的，就相当于可以在结构体之后给他起一个别名
{
	int val;    //存储树的值
	int num;    //存储树结点的值出现的个数
	struct bstNode *left_child, *right_child;//左孩子和右孩子

}*bst;

bool bstInsert(bstNode *&p, int e) {  //向树中插入结点
	if (p == NULL) {   //判断是否树为空
		p = (bstNode *)malloc(sizeof(bstNode ));  //动态申请内存
		//p = new bstNode;
		p->val = e;
		p->left_child = p->right_child = NULL;
		p->num = 0;
	}
	if (e == p->val) {
		p->num++;     //若插入的元素值同结点的元素值相同，则出现次数num+1；
		return false;
	}
	if (e < p->val) {
		return bstInsert(p->left_child, e);
	}
	if (e > p->val) {
		return bstInsert(p->right_child, e);
	}

}

void buildTree(bstNode *& T,int a[],int k) {//递归建树
	T = NULL;
	int q = 0;
	for (q = 0; q < k; q++) {
		bstInsert(T, a[q]);
	}
}

void midSort(bst T) {   //中序遍历输出，也就是左中右的顺序，这里也采取了递归的方式
	if (T) {
		midSort(T->left_child);

		cout << T->val << ":" << T->num << endl;

		midSort(T->right_child);

	}

}

int main() {
	int m, n;
	cin >> m >> n;
	int a[t];
	for (int i = 0; i < m*n; i++) {
		cin >> a[i];
	}
	bst tnode;
	buildTree(tnode , a,m*n);
	midSort(tnode);
	return 0;
	
}

class Solution {
public:
    int reverse(long int x) {
        int max =2147483647;  //根据题目所说的32位有符号整数定的范围
        int min =-2147483648;
        int rev = 0;   //记录反转后的数
        while (x != 0) {
            int pop = x % 10;
            x /= 10;
            if (rev > max/10 ) return 0;
            if (rev < min/10 ) return 0;
            rev = rev * 10 + pop;
        }
        return rev;
    }
    
};
