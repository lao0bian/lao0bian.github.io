docker images 列出本地主机上的镜像

  • -a: 列出本地的所有镜像
  • -q：只显示镜像ID
  • –digests:显示镜像的摘要信息
  • –no - trunc :显示完整的镜像信息### docker

docker search 某个xxx镜像的名字（查找镜像）

  • –no-trunc：显示完整的镜像描述
  • -s:列出收藏数不小于指定值的镜像
    eg：
  • –automated：只列出automated build类型的镜像

docker pull 某个xxx镜像的名字（下载镜像）

  • docker pull 镜像名字[:TAG]
  • docker pull tomcat 等价于 docker pull tomcat:latest

docker rmi 某个镜像名字/ID （删除镜像）

  • 删除单个：docker rmi -f 某个镜像名字/ID （强删）
  • 删除多个：docker rmi -f 镜像名:TAG 镜像名:TAG
  • 删除全部：docker rmi -f ${docker images -qa}
